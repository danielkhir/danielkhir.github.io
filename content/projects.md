Here are some of the stuff I've worked on.

### dash

A personal university dashboard. Project for the Web Programming II module at THD.

[Live Demo](https://dash.danielmuhd.dev)

Features:
- daily information pulled from various APIs (OpenWeather, OpenMensa, OpenStreetMap)
- JWT authentication
- packaged with Docker Compose
- deployed with GitLab CI to a DigitalOcean Droplet

Technologies used: Vue, Typescript, Express, MongoDB, Docker, Gitlab CI, Nginx, DigitalOcean

### Offline Speech Assistant

A voice assistant for offline use in cars. Group project for the AI Project module at THD.

Contributions:
- expert engineer for text-to-speech and intent classification subtasks
- containerized assistant submodules
- refactored voice recognition pipeline
- wrote audio manipulation functions

Technologies used: Python, Rasa, Docker, Git, FFmpeg
